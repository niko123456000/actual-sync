Account mapping

Pair Redbark accounts with Actual Budget accounts, then copy the mapping into your config or add-on.

1. Export account data

With a .env set, run one of:

./scripts/setup-account-ids.sh --export

or from repo: pnpm dev -- --export-accounts

or Docker: docker run --rm --env-file .env -v actual-sync-data:/app/data <image> --export-accounts

Copy the full JSON output, then paste it below. Or open this page from the add-on Web UI to load from your config.

2. Paste JSON

3. Pair accounts

For each Redbark account, choose the matching Actual Budget account.

4. Copy mapping

Paste this into Account mapping in your config or Home Assistant add-on Configuration.
